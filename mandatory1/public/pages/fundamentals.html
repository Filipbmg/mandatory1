Declaring Variables

In JavaScript there are three ways of declaring variables: var, let and const.
Although it is still possible to use it, it is no longer recommended to use var for anything JavaScript related.
The reason why var is outdated at this point, is because unlike let and const, var does not have block scope.
This means var variables are not limited to the block they're defined in, which can quickly result in messy code and undesirables outcomes if you're not careful.

The difference between let and const variables is that let variables can be reassigned, while const variables cannot.
A good rule of thumb to avoid any confusion, is to use const whenever possible, and let for any other scenario.
