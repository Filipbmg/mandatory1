Node.js

Node.js is a JavaScript runtime environment which can execute JavaScript code outside of the web browser.
This broadens the scope for what is feasible to do with JavaScript. Server-side development being a particularly noteworthy example.

The npm(Node Package Manager) is the designated package manager for Node.js.
It enables the developer to easily install, manage and update dependencies for their Node.js projects.

Express.js

Express.js is a lightweight web application framework.
As a lightweight framework, it provides a solid set of baseline features,
while simultaniously giving the developer the flexibility to structure their project as they see fit.
Express.js can be used to build web and mobile applications.
In our case we also use it for server development, as it drastically simplifies the process.

Commands and Code

Initializing a Node.js project is incredibly easy. You initialize a Node.js project with this simple terminal command: 


                npm init //Include -y at the end for default settings

Installing Express.js is just as simple: 

                npm install express

Once your project is initialized, you can initialize an app with the following lines of code in your project: 

                import express from "express"; //Imports the express module
                const app = express(); //Creates an instance of an Express application
                
                app.listen(port); //Server launch

There is of course much more you can do with Node.js and Express.js, but this is all it takes to get started with both.
